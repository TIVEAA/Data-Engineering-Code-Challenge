# Configuration Files

This directory contains configuration files used by the data processing pipeline.

## Files

- `tables_config.json`: Contains configuration settings for the tables, including paths and data types.
- `output_config.json`: Contains configuration settings for the output files, including paths and formats.

## Usage

These configuration files are loaded by the pipeline to determine how data should be processed and where results should be saved. Ensure that the paths and settings in these files are correct before running the pipeline.
