# Source Code

## Overview
This folder contains the source code for the challenge. Each script is responsible for a specific part of the data processing pipeline.

## Scripts
- `data_preparation.py`: Prepares the data by loading, validating, and cleaning the datasets.
- `data_transformation.py`: Transforms the data to generate insights and enrich the datasets.
- `data_export.py`: Exports the transformed data to the desired formats.
- `utils.py`: Contains utility functions used across the scripts.
